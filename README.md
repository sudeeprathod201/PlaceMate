# PlaceMate

## Description
PlaceMate is a Student Placement Management System that manages placement-related data. The system stores student details, company information, placement year, salary packages, job roles, and placement status using Google Sheets as the backend database.

## Features
- Student data management
- Company details tracking
- Placement year records
- Salary package tracking
- Placement status monitoring
- Simple and user-friendly interface
- Data storage using Google Sheets

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets

## How It Works
1. Users enter placement details through the application.
2. Data is processed using Google Apps Script.
3. Information is stored and managed in Google Sheets.
4. Records can be viewed, updated, and tracked easily.
